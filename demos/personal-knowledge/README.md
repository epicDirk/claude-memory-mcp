# 🧠 Personal Knowledge Manager

> **Your AI builds connections between your notes that you'd never find manually.**

## The Scenario

A polymath's 6 months of learning across 4 domains — philosophy, finance, biology, and strategy. 10 books, 15 cross-domain concepts, personal insights that connect ideas separated by centuries. The second brain that actually thinks.

## What Dragon Brain Finds

### 1. Cross-Domain Patterns
```
search_memory("concepts that appear across multiple books")
```
**Result:** Power laws (Scale + Antifragile + Pareto), feedback loops (systems theory + biology + economics), optionality (Taleb + Sun Tzu + portfolio theory). Patterns that span domains and centuries.

### 2. Concept Mapping
```
get_hologram("optionality")
```
**Result:** Which books discuss it, who coined it, where it's been applied, connected concepts (antifragility, barbell strategy, asymmetric bets), and personal projects that use it.

### 3. Intellectual Disagreements
```
search_memory("where does Taleb disagree with Kahneman")
```
**Result:** Specific contradictions between their frameworks — Taleb's critique of prospect theory's treatment of tail risk, Kahneman's skepticism of narrative-driven reasoning.

### 4. Surprise Connections
```

```
**Result:** Sun Tzu → strategic deception → information asymmetry → Munger's mental models → inversion thinking. A 2,500-year intellectual thread connecting a Chinese general to a modern investor.

### 5. Missing Links
```

```
**Result:** *"Sun Tzu's 'strategic positioning' is semantically similar to Taleb's 'barbell strategy' — both are about asymmetric positioning with capped downside. No graph link exists."*
