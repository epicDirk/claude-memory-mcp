## The Scenario

**Pacific Rim Shell Network** — tracking corporate ownership across jurisdictions. Shell companies in Delaware, Cayman, Singapore, and Cyprus. Nominee directors. Circular money flows. The kind of investigation where one missed connection changes everything.

## What Dragon Brain Finds

### 1. Financial Pattern Detection
```
search_memory("circular money flows")
```
**Result:** Finds the loop: Pacific Trading → Emerald Holdings → Singapore intermediary → back to Pacific Trading. $4.7M moving in a circle across three jurisdictions — a classic layering pattern.

### 2. Red Flag Detection
```
search_memory("companies sharing registered addresses")
```
**Result:** Three companies registered at 45 Harbor View, George Town — all incorporated within 2 weeks. Same registered agent. Same nominee director. Dragon Brain clusters the red flags.

### 3. Hidden Ownership Chains
```
traverse_path(director_y_id, pacific_trading_id)
```
**Result:** Traces the ownership through nominee services: Director Y → Atlas Corporate Services (nominee) → Pacific Trading Ltd. The beneficial ownership chain that corporate registries are designed to obscure.

### 4. Temporal Clustering
```
query_timeline(start="2024-03-01", end="2024-04-01")
```
**Result:** 5 companies incorporated within 4 weeks across 3 jurisdictions. Temporal clustering that suggests coordinated activity — not coincidence.

### 5. Lead Discovery
```
semantic_radar(director_y_id)
```
**Result:** *"Director Y semantically linked to a Cyprus company — no graph path. Worth investigating."* A connection the journalist hasn't made yet.

## 💡 The Holy Shit Moment
